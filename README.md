# RateMesh

RateMesh is a fresh protocol and app direction for open prediction-based ratings.
It reuses the strongest parts of the existing Curyo codebase where that is
practical, while replacing Self.xyz, the faucet, and binary up/down voting with
an open Base-native rater network built around predicted final ratings, capped
transferable Mesh Reputation (`MREP`), MREP lock redistribution, decentralized
governance, frontend staking/incentives, AI rater support, and cluster-aware
USDC payouts.
