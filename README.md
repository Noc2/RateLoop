# Curyo

Curyo is continuing as an open prediction-based ratings protocol and app. The
new design keeps the current Curyo brand, logo, and visual identity while
replacing Self.xyz, the faucet, and binary up/down voting with a Base-native
rater network built around predicted final ratings, capped transferable Curyo
Reputation (`CREP`), CREP lock redistribution, decentralized governance,
frontend staking/incentives, AI rater support, and cluster-aware USDC payouts.
