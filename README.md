# RateMesh

RateMesh is a fresh protocol and app direction for open prediction-based ratings.
It reuses the strongest parts of the existing Curyo codebase where that is
practical, while replacing the Self.xyz, faucet, legacy HREP staking, and binary
up/down voting model with an open rater network built around prediction,
capped transferable reputation, decentralized governance, frontend incentives,
and cluster-aware USDC payouts.
