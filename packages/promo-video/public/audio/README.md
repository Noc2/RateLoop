# Promo video audio assets

- `vo-*.m4a` - per-beat voiceover generated by
  `scripts/generate-openai-voiceover.mjs` with OpenAI TTS. The legacy
  `scripts/generate-voiceover.zsh` script remains as an offline fallback.
- `music.m4a` - upbeat 104 BPM synth bed generated by
  `scripts/generate-music.mjs`. It uses procedural drums, bass, plucks, pads,
  and small risers with no samples, so rights are clean.
