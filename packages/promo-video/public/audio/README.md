# Promo video audio assets

- `vo-*.m4a` - per-beat voiceover generated by
  `scripts/generate-openai-voiceover.mjs` with OpenAI TTS. The legacy
  `scripts/generate-voiceover.zsh` script remains as an offline fallback.
- `music.mp3` - "Travel Promo" by The_Mountain, downloaded from Pixabay:
  https://pixabay.com/music/main-title-travel-promo-158075/
  The track is listed as free for use under the Pixabay Content License.
- `music.m4a` - legacy procedural 104 BPM synth bed generated by
  `scripts/generate-music.mjs`; kept as an offline fallback.
